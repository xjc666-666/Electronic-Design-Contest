# STM32F407 电赛基础模板

这是一个为全国大学生电子设计竞赛（全国电赛）准备的 STM32F407 基础开发工程模板。工程已经配置好基础的环境，并且包含了大量在电赛中经常使用的外设、信号链模块（如 ADC、DAC、DDS、PLL 等）的驱动代码。

## 驱动模块介绍

工程的 `Drive` 文件夹下包含了以下硬件模块的驱动源码（`.c` 和 `.h`）：

### 1. 模数 / 数模转换模块 (ADC & DAC)
- **内部资源**：
  - `ADC.c`：STM32F407 内部 ADC 驱动
  - `DAC.c`：STM32F407 内部 DAC 驱动
- **高精度 / 高速 ADC**：
  - `AD7608.c`：8通道、18位、真双极性同步采样 ADC
  - `ADS805.c`：12位、20MSPS 模数转换器
  - `ADS8688.c`：16位、8通道、集成模拟前段的 SAR ADC
- **高精度 DAC**：
  - `DAC8560.c`：16位、极低毛刺、带参考电压的 DAC
  - `TLV5638.c`：12位、双通道串行 DAC
- **多功能混合模块**：
  - `AD5593.c`：带内部参考的8通道、12位、可配置 ADC/DAC/GPIO 模块

### 2. 信号发生 / 频率合成模块 (DDS & PLL)
- **直接数字频率合成器 (DDS)**：
  - `AD9833.c`：低功耗、可编程波形发生器 (正弦波、三角波、方波)
  - `AD9954.c`：400 MSPS、14位、高频 DDS
  - `AD9959.c`：4通道、500 MSPS、独立相位和幅度控制 DDS
- **锁相环 / 频率合成器 (PLL)**：
  - `ADF4351.c`：带集成 VCO 的宽带分数 N 分频 / 整数 N 分频合成器
  - `HMC830.c`：宽带小数 N 分频 PLL
  - `LMX2571.c`：低功耗、高性能宽带频率合成器
  - `PLL.c`：通用锁相环控制驱动
- **时钟发生器**：
  - `SI5351.c`：I2C 可编程任意频率时钟发生器

### 3. 射频与阻抗分析模块 (RF & Impedance)
- **数控衰减器 (DSA)**：
  - `PE4302.c`：高线性度、6位数字步进衰减器 (0.5dB 步进，最大 31.5dB)
  - `DAT_31R5_SPpos.c`：串行控制步进射频衰减器
- **阻抗/网络分析**：
  - `AD5933.c`：1 MSPS、12位阻抗转换器 / 网络分析仪
- **其他**：
  - `PhaseDetect.c`：相位检测模块控制代码

### 4. 基础外设与系统模块 (Peripherals)
- **通信接口**：
  - `I2C_Software.c`：软件模拟 I2C 驱动
  - `USART.c`：串口收发与配置
- **人机交互**：
  - `OLED.c` & `OLED_Font.h`：常见 0.96寸 OLED 屏幕驱动及字库
  - `Keyboard.c`：按键及矩阵键盘驱动
  - `led.c`：基础 LED 指示灯控制
- **控制与基础外设**：
  - `delay.c`：精确延时函数
  - `GPIO.c`：基础 IO 端口配置
  - `Timer.c`：定时器配置（如产生固定频率中断、PWM 等）
  - `DMA.c`：直接存储器访问控制器配置
  - `Relay.c`：继电器控制

## 使用说明

工程基于 Keil MDK 开发环境，采用标准库 (Standard Peripheral Library) 编写。您可以直接打开 `Project` 目录下的 `.uvprojx` 工程文件进行编译和下载。所有驱动均已经过初步测试，在实际电赛应用时可根据具体的引脚分配在对应的驱动头文件 (`.h`) 中修改 IO 宏定义。
